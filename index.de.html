Server Dienste

Was für Dienste

Ich, das heißt Tim Schumacher betreibe einen Server der zur dezentralen Kommunikation über das XMPP-Protokoll genutzt werden kann. Ich versuche den Server nach besten Wissen und gewissen zu betreiben. Auf folgenden Domains ist mein Server erreichbar:

  • boese-ban.de
  • datenknoten.me
  • kaoskinder.de

Ich setze als Server-Software Prosody in der Version 0.10 ein. Folgende Komponenten Sind aktiv:

  • Multi User Chat (muc)
  • HTTP-Upload (100 MB max pro User, 20 MB maximal pro Datei, Dateien werden 30 Tage aufbewahrt).

Um dir ein neues Konto zu beziehen must du das Registrierungsformular benutzen.
